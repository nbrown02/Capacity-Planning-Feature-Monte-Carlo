# Security Policy

## Overview

The Capacity Planning, Right-sizing & Epic Monte Carlo template is provided **free of charge** and is designed to help teams visualize their workflow metrics/delivery dates more effectively. This template is a **static configuration file** and does **not** include any code that executes on Jira/ADO instances or external systems.

## Data Privacy and Security

- The template does **not** collect, transmit, or store any user or company data.
- There are **no** external dependencies, scripts, or integrations that could expose sensitive information.

## Scope and Liability

- This template is provided **as-is**, with **no warranties or guarantees** regarding its functionality or compatibility with all Jira/ADO environments.
- Users are encouraged to review the template before applying it to their Jira/ADO instance.
- The maintainers of this repository do **not** provide direct support but welcome community contributions and discussions.

## Reporting Security Concerns

If you identify any potential security issues related to this template, please open an issue in this repository or contact us via [GitHub Issues](https://github.com/nbrown02/Capacity-Planning-Feature-Monte-Carlo/issues).
